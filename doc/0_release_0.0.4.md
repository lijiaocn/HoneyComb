## 摘要

首先，到AuthnAuthz中准备证书和授权策略; 

然后到Compile中完成组件的编译; 

最后，将组件和证书拷贝到要对应到机器上，修改配置文件，启动应用。

## AuthnAuthz

在相应目录下建立以目标机器的IP或者域名命名的目录。运行gen.sh，根据提示完成操作。

## Compile

进入相应的目录下，运行同名的".sh"脚本，完成编译。
